# Security and Compliance

## 公開リポジトリに入れてはいけないもの

- パスワード
- Cookie
- セッション情報
- APIキー
- 個人アカウントの内部情報
- 未公開の顧客情報
- 有料記事の本文
- 他社画像やスクリーンショットの無断転載素材

## Codexに渡してよいもの

- 投稿ルール
- 投稿テンプレート
- SNS運用ノウハウ
- 公開情報のURL
- 画像生成プロンプト
- 手順書
- チェックリスト

## 投稿前確認が必要な操作

- Threadsへの投稿
- X、Instagram、TikTok、noteなど外部サービスへの投稿
- GitHubリポジトリの公開
- ファイル削除
- 既存投稿の削除
- 課金が発生するAPI利用

## ニュース利用

- 公式発表や信頼できる報道を確認する
- 記事本文を長く引用しない
- 引用する場合は短くし、URLを添える
- 未確認情報は投稿しない
- 推測や意見は事実と分けて書く

## ブランド安全性

- 読者を煽りすぎない
- 不安を過度に刺激しない
- 投資助言や医療助言など高リスク領域では断定しない
- AIツールの性能を誇張しない
- 投稿後に誤りが見つかったら訂正する
