# 競合アカウント分析ルール

## 目的

競合の投稿傾向を把握し、差別化された自分の投稿テーマ・構成を設計する。

## 分析手順

1. 指定されたURLのThreadsアカウントをブラウザで開く
2. 最新の公開投稿を上から確認する（目安：10〜20件）
3. 以下の項目を記録する

## 記録項目

- よく使うテーマ（3〜5個）
- フックの型（疑問形 / 断言 / 数字 / 共感など）
- 投稿構成（単発 / ツリー / 比率）
- 特徴的な言い回しや口癖
- 話題にしていない領域（差別化ポイント）

## 禁止事項

- 投稿文をそのまま、または少し変えて流用しない
- 分析結果はあくまで傾向把握の参考にとどめる
- 自動的な大量取得はしない（手動閲覧の範囲で行う）
- セッション情報やCookieをClaude Codeに渡さない
