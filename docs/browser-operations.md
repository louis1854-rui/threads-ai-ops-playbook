# Chrome/Threads操作手順

## 前提

ユーザー自身のChromeでThreadsにログインしておく。Codexはログイン情報を扱わず、ログイン済みブラウザを操作する。

## 投稿手順

1. ChromeのThreadsタブを探す
2. なければ `https://www.threads.com/` を開く
3. 投稿作成ボタンを押す
4. 投稿本文を入力する
5. ツリー投稿の場合は「スレッドに追加」または同等ボタンでリプを追加する
6. 画像投稿の場合は画像を添付する
7. 投稿前に画面内容を確認する
8. 投稿ボタンを押す
9. プロフィールまたは投稿ページでURLを取得する

## 既知の失敗と回避策

### テキスト入力が失敗する

ブラウザ自動操作で入力できない場合がある。その場合はOSのクリップボードへ本文を入れ、Chrome上で貼り付ける。

### ファイルアップロードが拒否される

ブラウザ自動操作のファイル選択が許可されない場合がある。その場合は画像をOSのクリップボードに入れ、Threads投稿欄へ貼り付ける。

### 投稿ボタンが見つからない

画面幅、言語設定、ThreadsのUI変更でボタン名が変わる場合がある。候補。

- 投稿
- Post
- Share
- スレッドに追加
- Add to thread

### 投稿直後にURLが取れない

プロフィールページを更新し、最新投稿を開いてURLを取得する。

## チェックリスト

- 投稿先アカウントが正しい
- 本文に誤字がない
- 日付が正しい
- 出典と内容が一致している
- 画像が添付されている
- ツリー順が崩れていない
- 投稿後URLを記録した
