# 投稿パターン集

## AIニュース速報

```text
今日のAIニュースで押さえるべきはこの3つです。

1. {ニュース1}
2. {ニュース2}
3. {ニュース3}

特に重要なのは{一番重要な変化}。
理由は、{読者への影響}だからです。
```

## ツリー1投稿目

```text
今日のAIニュース、表面だけ見ると「また新機能か」で終わります。

でも実務目線だと、かなり大きい変化が起きています。
ポイントは3つ。
続きで整理します。
```

## リプ構成

```text
1つ目。
{事実}

これは{意味}という点で重要です。
```

```text
2つ目。
{事実}

個人や小規模事業者にとっては、{使い方}に直結します。
```

```text
3つ目。
ここは注意です。

{制約やリスク}
なので、今すぐ全部置き換えるより、{現実的な使い方}が良さそうです。
```

```text
まとめると、
今日の変化は{短い結論}です。

まず試すなら{具体的な一歩}。
この視点で追うとAIニュースがかなり実務に変わります。
```

## 図解投稿文

```text
今日のAIニュースを1枚に整理しました。

細かい機能名より大事なのは、
「AIがどこに組み込まれていくか」です。

保存用にどうぞ。
```

## 反応を取りやすい問い

```text
あなたの仕事で一番置き換わりそうなのはどこですか？
```

```text
この中で一番使ってみたいのはどれですか？
```

```text
AIニュース、速報と実務解説ならどちらが読みたいですか？
```

## NG例

- 出典なしで「最新」と言い切る
- 一次情報を確認せずに噂を投稿する
- 専門用語だけで読者を置いていく
- 画像に文字を詰め込みすぎる
- 投稿ごとにトーンが大きく変わる
