# Codex実行プロンプト

下記をCodexに貼り付けると、このリポジトリの運用ルールに沿ってThreads投稿を作成できます。

```text
このリポジトリのAGENTS.md、docs/automation-workflow.md、docs/sns-growth-playbook.md、docs/content-patterns.md、docs/browser-operations.md、docs/security-and-compliance.mdを読んでください。

今日のAI最新情報をWebで調査し、Threads向けに以下を作ってください。

1. 1投稿目で続きを読みたくなるツリー投稿
2. リプ3〜5件
3. 同じテーマの図解投稿1枚
4. 投稿文、出典URL、注意点

投稿前に必ず確認を取ってください。
私が「投稿して」と明示したら、Chromeのログイン済みThreadsを操作して投稿してください。
ログイン情報やCookieは要求しないでください。
投稿後はURLと改善メモを記録してください。
```

## 日次運用プロンプト

```text
このリポジトリのルールに従って、今日のAIニュースをThreads用に投稿準備してください。
対象読者はAIで収益化・業務効率化したい個人と事業者です。
ツリー投稿は3〜5リプ、図解投稿は1枚、投稿前確認ありで進めてください。
```

## 週次分析プロンプト

```text
このリポジトリのSNS運用ルールに従って、直近1週間の投稿ログを分析してください。
伸びた投稿の共通点、伸びなかった投稿の改善案、次週の投稿テーマ案を出してください。
```
