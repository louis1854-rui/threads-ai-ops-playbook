# Image Prompt Template

## 1080x1080 図解

```text
Create a square 1080x1080 Japanese infographic for Threads.

Topic: {topic}
Audience: {audience}
Main message: {main_message}

Layout:
- Strong title at the top
- 3 to 5 clear sections
- Simple icons or abstract UI elements
- High contrast
- Plenty of white space
- Large readable Japanese text

Style:
- Modern editorial infographic
- Clean tech media design
- Professional but approachable
- No fake product screenshots
- No copyrighted logos
- No tiny text
- No dense paragraphs

Text to include:
Title: {title}
Point 1: {point_1}
Point 2: {point_2}
Point 3: {point_3}
Point 4: {point_4_optional}
Footer: {footer}
```

## 投稿文と合わせるコツ

- 画像は保存したくなる整理にする
- 投稿文は画像を見る理由を作る
- 画像内の文章を投稿文でそのまま繰り返さない
- 画像は結論、投稿文は文脈と解釈にする
